# Digital Maturity Check

Check di maturità digitale operativa per PMI italiane.

## 🚀 Deploy

Questo sito è live su GitHub Pages.

## 📋 Struttura

- `index.html` - Pagina principale con check e generazione PDF
- `example-pages-data.js` - Immagini del PDF esempio per il modal
- `dataset.json` - Domande e KPI del questionario
- `assets/` - Logo e risorse grafiche
